최종적으로 완성된 **Geo Challenge(위성 지도 지리 맞히기 게임)**의 스펙과 구현 내용을 프롬프트 형태로 정리해 드립니다. 이 내용은 나중에 비슷한 프로젝트를 수행하거나 기록으로 남기실 때 활용하시기 바랍니다.

[최종 결과물: Geo Challenge 게임 프로젝트 요약]
1. 프로젝트 개요
목적: 전 세계 위성 지도를 보고 해당 위치의 국가 또는 도시 이름을 맞히는 서바이벌 웹 게임.
기술 스택: HTML5, Vanilla CSS (Glassmorphism), JavaScript (ES6+), MapLibre GL JS (오픈소스 지도 라이브러리).
지도 데이터: Esri World Imagery (위성 영상) + Esri World Boundaries and Places (국가 경계선 및 지명 라벨).
2. 핵심 게임 로직
단계별 난이도:
초기(110단계): 국가 단위 위성 지도 (Zoom 35).
심화(11단계~): 도시 단위 상세 지도 (Zoom 11~12).
타이머 시스템: 30초에서 시작하여 매 스테이지 성공 시마다 0.1초씩 줄어드는 동적 타이머 (최소 2초).
기회(Life) 시스템: 총 3번의 기회(❤️) 제공. 3번의 오답 시 게임 종료.
힌트 시스템: 오답 시 정답의 첫 글자만 노출하여 도전 의식 고취 (예: "프...").
정답 판정: 한글과 영어(대소문자 무관) 모두 지원하는 다중 정답 처리.
3. 사용자 기록 및 순위 시스템
사용자 식별: 게임 시작 시 이메일 주소를 입력받아 기록 관리.
데이터 저장: 브라우저 localStorage를 사용하여 서버 없이 순위 저장.
중복 제거: 동일한 이메일 주소에 대해서는 최고 점수(High Score) 하나만 순위에 노출되도록 자동 필터링.
리더보드: 게임 종료 시 전체 사용자 중 상위 10위 명예의 전당 및 나의 현재 순위 표시.
4. UI/UX 디자인 특징
디자인 테마: 심해의 인디고와 에메랄드 그린 그라데이션을 사용한 현대적 다크 모드.
글래스모피즘: 반투명한 블러 배경과 가느다란 테두리를 사용한 카드 디자인.
반응형 레이아웃: 지도가 화면을 꽉 채우도록 구성하고, 맵 리사이징 최적화를 통해 로딩 시 잘림 현상 해결.
애니메이션: 틀렸을 때 화면이 흔들리는 쉐이크 효과 및 스테이지 전환 시 부드러운 지도 이동(flyTo).
이 프롬프트는 사용자가 요청한 "재밌고, 어렵고, 경쟁적인" 요소들을 모두 포함하고 있으며, 특히 지리 교육과 게임을 결합한 에듀테인먼트로서의 가치를 극대화하도록 설계되었습니다. 게임 실행을 위한 모든 소스 코드는 현재 작업 디렉토리의 geo_guessing_game 폴더에 준비되어 있습니다. 🌍🧭✨
