🎮 구구단 챌린지 (Multiplication Challenge) 
사용자의 순발력과 연산 능력을 테스트하는 중독성 있는 수학 퀴즈 게임입니다.

1. 프로젝트 개요
목적: 제한된 시간 내에 임의로 생성되는 구구단 문제를 정답 처리하고 최고 점수에 도전하는 게임.
기술 스택: HTML5, Vanilla CSS, Pure JavaScript (순수 프론트엔드 기술만 사용).
디자인 컨셉: 사이버네틱 다크 모드, 네온 블루/퍼플 그라데이션, 글래스모피즘 UI.

2. 게임 메커니즘
문제 생성: 2단부터 9단까지의 문제가 무작위로 출제되며, 스테이지가 높아질수록 난이도가 미세하게 조정됨.
초고속 진행: 정답을 입력하는 즉시 별도의 확인 버튼 없이 다음 문제로 자동 전환되는 사용자 경험(UX) 최적화.
타이머 시스템:
초기 스테이지: 10.0초 제공.
난이도 상승: 성공할 때마다 제한시간이 0.1초씩 감소.
시각적 효과: 시간이 줄어들수록 타이머 바의 색상이 초록에서 빨강으로 변화.
실패 조건: 타임오버 시 즉시 게임 종료 및 결과 화면 전환.

3. 기록 및 경쟁 시스템
사용자 인증: 이메일 주소를 기반으로 개인 기록 식별.
로컬 랭킹: localStorage를 활용하여 브라우저에 순위 데이터 상시 저장.
최고점 유지: 동일 이메일 사용자 중복 시 가장 높은 점수(High Score) 하나만 리더보드에 적용.
최종 스태츠: 종료 시 달성한 스테이지, 최종 점수, 내 순위를 한눈에 확인 가능.

4. 시각 및 인터랙션 디자인
글래스모피즘(Glassmorphism): 배경이 비치는 반투명 카드 UI와 backdrop-filter 효과 적용.
애니메이션:
스테이지 진입 시 카드가 부드럽게 떠오르는 효과.
오답 또는 게임 종료 시 화면이 흔들리는 Shake 애니메이션.
타이포그래피: 'Outfit' 및 'Noto Sans KR' Google Fonts를 사용하여 가독성과 세련미 확보.
이 게임은 이후에 만든 Geo Challenge의 기반이 된 프로젝트로, 빠른 반응 속도와 현대적인 인터페이스가 특징입니다. 관련 소스 코드는 multiplication_game 폴더 내에 보관되어 있습니다. 🧠⚡
